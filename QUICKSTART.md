# Quick Start Guide

Get battle-tested website copy in 5 minutes.

## Install

```bash
npx @psrmanju/website-copy-pro
```

## Use

In Claude (CLI or Claude.ai):

```
/website-copy-pro
```

## Provide Info

Claude will ask you about:

1. **Your product** - What it is in one sentence
2. **The problem** - Specific pain it solves
3. **Target audience** - Who desperately needs this
4. **What makes it different** - Actual differentiator
5. **Conversion goal** - What action you want

## Get Your Copy

Claude outputs:

- Hero section (headline, subheadline, CTA)
- Problem agitation (4 pain points with emojis)
- Solution steps (how it works in 3 steps)
- Benefits with proof (3 transformations)
- Social proof templates (testimonial types)
- FAQ (5-6 objections handled)
- Final CTA section
